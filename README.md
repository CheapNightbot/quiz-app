- TypeScript

## Getting Started

Ready to quiz? Follow these easy steps:

1. **Clone the repo** (if you haven't already):
   ```bash
   git clone https://github.com/CheapNightbot/quiz-app.git
   cd quiz-app
   ```

2. **Install dependencies**:
   ```bash
   deno install
   ```

3. **Run the dev server**:
   ```bash
   deno task dev
   ```
   Open your browser to [`http://localhost:5173`](http://localhost:5173) and start quizzing!

4. **Build for production** (when you're ready to share the fun):
   ```bash
   deno task build
   ```
